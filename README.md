# Azure Bicep Examples

Este repositório reúne exemplos práticos e didáticos de **Bicep** para implementação de recursos no Microsoft Azure.  
Cada pasta contém um `main.bicep` simples, direto e ideal para estudo, demonstrações, workshops e aulas.

O objetivo é fornecer uma base clara para quem está a aprender **Infraestrutura como Código (IaC)** com Azure.
